# buession-security-crypto 参考手册


数据加密、解密类库，支持：MD5、SHA1、SHA256、SHA512、BASE64 以及 Discuz 加密算法等等接口。


---


### 安装

```xml
<dependency>
    <groupId>com.buession.security</groupId>
    <artifactId>buession-security-crypto</artifactId>
    <version>x.x.x</version>
</dependency>
```
